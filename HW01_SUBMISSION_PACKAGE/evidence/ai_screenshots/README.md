# AI Output Screenshots

Thư mục này dùng để lưu **ảnh output AI thật**. Không tạo ảnh giả và không dùng ảnh AI-generated. Ảnh phải được chụp từ cuộc trò chuyện AI thật mà bạn đã dùng trong quá trình làm HW01.

## Cách lấy ảnh output AI thật

1. Mở lại cuộc trò chuyện ChatGPT/Codex thật đã dùng để làm bài.
2. Kéo tới đoạn prompt và output AI liên quan.
3. Chụp màn hình bằng `Cmd + Shift + 4` trên macOS.
4. Lưu ảnh vào thư mục này.
5. Đổi tên ảnh đúng theo danh sách bên dưới.

## Các ảnh AI output cần bổ sung

Các file dưới đây được dùng trong `AI_02_Audit_Report.md`:

- `A1_requirement1_jobs_ai_output.png`

- `A3_mouse_test_cases_ai_output.png`

Trong đó `A3_mouse_test_cases_ai_output.png` cũng được `HW01_Main_Report.md` nhúng tự động ở Requirement 3.

Ảnh A3 nên thể hiện output AI ban đầu tạo test cases cho chuột máy tính. Nếu có thể, ảnh nên thấy cả prompt kiểu:

```text
Generate 15 test cases for testing a computer mouse. Include objective, input, steps, expected result, actual result, and verdict.
```

Các ảnh này không thay thế prompt log, nhưng giúp chứng minh AI Audit Report dựa trên output thật.
